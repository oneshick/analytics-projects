# Data Analysis Projects

Этот репозиторий содержит два проекта для работы с данными: генерация, хранение и визуализация.

## Структура репозитория

### `shop/`
Мини-система сбора данных и анализа:
- Генератор данных на Go (`generator/`)
- SQL-схема для PostgreSQL (`db/init.sql`)
- Jupyter Notebook для анализа (`notebooks/analysis.ipynb`)
- Docker Compose для развёртывания всех сервисов (`docker-compose.yml`)

Проект предназначен для демонстрации end-to-end процесса: от генерации данных до анализа в Jupyter.

### `redash/`
Проект с Redash для визуализации данных и построения дашбордов:
- Настройки и скрипты для развёртывания Redash
- Интеграция с PostgreSQL/MySQL
- Возможность создавать графики, таблицы и дашборды для аналитики

## Основные технологии
- Go — генерация данных
- PostgreSQL — база данных
- Docker и Docker Compose — контейнеризация сервисов
- Jupyter Notebook — интерактивный анализ данных
- Redash — визуализация данных и дашборд
